```diff
+ Fast multithreaded LAN reconnaissance tool

```

---

## 🚀 Features

* ⚡ Multithreaded scanning engine
* 🌐 LAN device discovery
* 🧠 Hostname resolution
* 🧲 MAC address detection (ARP)
* 🔌 Common port scanner (21,22,80,443,445...)
* 🖥️ Device type fingerprinting
* 📊 Live progress visualization

